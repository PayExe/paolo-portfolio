Mémo : créer une image Docker pour ton projet Vite React

1. Crée un fichier Dockerfile à la racine du projet avec ce contenu :

# Étape 1 : build de l'app
FROM node:18 AS build
WORKDIR /app
COPY . .
RUN npm install
RUN npm run build

# Étape 2 : serveur statique
FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
EXPOSE 80

2. Construis l’image Docker :
docker build -t paolo-portfolio .

3. Pour donner l’image à ton pote :
- Soit tu la pushes sur Docker Hub (ou autre registry) :
  docker tag paolo-portfolio tonpseudo/paolo-portfolio
  docker push tonpseudo/paolo-portfolio
- Soit tu l’exportes en fichier :
  docker save paolo-portfolio > paolo-portfolio.tar
  Il pourra la charger avec :
  docker load < paolo-portfolio.tar

4. Pour lancer le conteneur :
docker run -p 80:80 paolo-portfolio

Voilà, prêt à être partagé !
